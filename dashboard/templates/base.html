Help

djProman is a project management tool developed by weavebytes.

Features
  1. Each project will have tasks and workers to handle them.
  2. Each task can have several comments by worker.
  3. Comments will have title, description and screenshots.
  4. There is a UI to add/edit/manage tasks.
  5. There is a UI to add/edit/manage comments.
